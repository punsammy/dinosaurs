6. Dinosaur names are hard to remember. Find the shortest dinosaur, and rename it Shortie.
SELECT MIN(length) FROM dinos;
 min
------
 0.08

SELECT species FROM dinos WHERE length = '0.08';
 species
-----------
 Delicatus

SELECT id FROM dinos WHERE length = '0.08';
 id
-----
 160

UPDATE dinos SET name  = 'Shortie' WHERE id = '160';

SELECT name FROM dinos WHERE id = '160';
 name

 Shortie

8. Rename the five longest dinosaurs The Famous Five.
SELECT length, id, name FROM dinos WHERE length > 0 ORDER BY length DESC, name ASC LIMIT 5;
 length | id  |      name
--------+-----+-----------------
     40 |  61 | Seismosaurus
     35 | 299 | Argentinosaurus

     30 | 280 | Brachiosaurus
